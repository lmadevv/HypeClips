# HypeClips

## How to run the server (testing):
```bash
export FLASK_APP=application.py
export FLASK_ENV=development
flask run
```

## How to run the front-end:
```bash
cd front-end
npm install # only on first run, or whenever dependencies are updated
npm run dev
```
